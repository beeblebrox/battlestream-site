Duos Support

Full partner tracking — hero, tier, triples, and last-seen board. Every chart is duos-aware with player/partner/paired tabs.

Live TUI Dashboard

Rich terminal interface with live game data, buff breakdowns, and one-key browser launch. Mouse-scrollable panels with resizable columns.

Web Dashboard

11 interactive ECharts — placement trends, hero performance, heatmaps, buff analysis, tribe win rates. Filter by mode, game count, date range.

Get Started in Seconds

Homebrew

macOS & Linux

brew tap beeblebrox/tap
brew install battlestream

Scoop

Windows

scoop bucket add battlestream https://github.com/beeblebrox/scoop-bucket
scoop install battlestream

Binary

All platforms

Download from Releases

From Source

Go 1.25+

git clone https://github.com/beeblebrox/battlestream
cd battlestream && go build -o battlestream ./cmd/battlestream

Then run it

# Discover your Hearthstone install
battlestream discover

# Start the daemon
battlestream daemon

# Open the TUI in another terminal
battlestream tui

# Or open the web dashboard
open http://localhost:8080/dashboard/
